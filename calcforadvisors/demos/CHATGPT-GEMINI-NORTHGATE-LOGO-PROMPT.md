# ChatGPT or Gemini prompt: Northgate Retirement Planning logo/avatar

Use this prompt in **ChatGPT** (with image generation) or **Gemini** to create a logo or avatar for the fictional firm Northgate Retirement Planning. The result will be used in the sample demo site, e.g. in place of the "NG" text mark in the banner.

---

**Prompt:**

```
Create a professional logo or avatar for a fictional financial services company called "Northgate Retirement Planning." The company offers retirement and income planning for individuals and families.

**Requirements:**
- **Format:** A square or nearly square image, suitable for use as a small logo/avatar (e.g. 256×256 pixels or similar). It will be displayed at about 56×56 pixels on a website, so it must read clearly when scaled down.
- **Content:** Either (a) a simple icon or symbol that suggests retirement, planning, growth, or stability (e.g. abstract mountain, upward trend, shield, or stylized "NG" monogram), or (b) a clean wordmark-style "NG" that could sit in a rounded square. No full company name required in the image—the name appears as text next to it.
- **Style:** Professional, trustworthy, B2B. Blues and grays preferred (or blue with a subtle accent). Clean and minimal—no busy details, no photographs. Suitable for a white or light background.
- **Use case:** This is for a sample/demo website only (fictional firm). The logo will sit in a rounded rectangle next to the heading "Retirement & income planning for individuals and families."

Output a single logo or avatar image. Save as northgate-logo.png or northgate-avatar.png.
```

---

**After you have the image:**  
Save it in the folder `calcforadvisors/demos/banners/` (e.g. `northgate-logo.png`). To use it in the Northgate demo page, replace the "NG" text mark with an `<img>` tag pointing to `banners/northgate-logo.png` and style it to fit the 56×56 rounded box (or ask for that code change).
