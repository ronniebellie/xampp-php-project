AI Budget Auditor

Pull your current-month YNAB categories, then ask GPT-4o for a concise spending review. API keys stay in this browser’s localStorage; YNAB is called directly, and the AI step uses a thin server proxy.

How it works

Paste your personal access token and OpenAI key below; both are saved in localStorage on this device. YNAB is fetched from your browser. The formatted budget summary and your OpenAI key are sent to /api/ynab_proxy.php, which forwards the chat request server-side (avoiding browser CORS limits). Keys are never stored on the server. Use last-used as the budget ID, or paste a specific budget UUID from YNAB.

🚀 Getting Started (60-Second Setup)

This tool is entirely serverless for your credentials — paste your keys below and they stay securely in this browser’s localStorage. Nothing is saved on the server.

  1. OpenAI API Key — Sign in and create a new secret key at platform.openai.com/api-keys, then paste it into the OpenAI field below.
  2. YNAB Personal Access Token — Open app.ynab.com/settings/developer, click New Token, copy the token, and paste it below. Note: If you normally use Google or Apple Sign-In to access YNAB, you will be prompted to quickly set a local account password first to unlock the “New Token” button.
  3. YNAB Budget ID — Leave this set to last-used to automatically fetch your most recently active budget. Only change it if you want to audit a specific budget UUID.
